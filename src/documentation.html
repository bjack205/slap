Documentation

To build the documentation you will need to install a few dependencies.

Linux

sudo apt-get install doxygen sphinx-doc -y
pip3 install sphinx-rtd-theme sphinx-sitemap breathe

MacOS

brew install doxygen sphinx-doc -y
pip3 install sphinx-rtd-theme sphinx-sitemap breathe
