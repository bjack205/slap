enum slap_ErrorCode slap_Map(Matrix mat, double (*function)(double))

Applies a function element-wise to every element in the matrix.

Applies mat[k] = function(mat[k]) for all elements

Parameters

  • mat – A valid matrix (can be dense or strided)

  • function – A function pointer that takes and returns a double
